## ⚙️ How to Use and Apply Your Gmail Filter XML File

This guide explains how to import the mailFilters.xml file into your Gmail account and how to apply the new rules to your past emails.

1. Import the XML File
   Open Gmail Settings: Click the Gear icon (⚙️) in the top right corner of Gmail, then select "See all settings".

2. Navigate to Filters: Go to the "Filters and Blocked Addresses" tab.

3. Initiate Import: Look near the top of the filter list and click the "Import filters" link.

4. Upload the File: Click "Choose File", select the XML file you created (e.g., mailFilters.xml), and then click "Open File".

5. Finalize Import: Click the "Create Filters" button. The new filters (for @backlog.jp and @cacoo.com) will now be active for all new incoming emails.
